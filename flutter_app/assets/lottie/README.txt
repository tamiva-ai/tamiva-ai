Drop Lottie JSON files here to have them replace the built-in procedural
progress animations. Expected filenames (all optional):

  generating.json   - shown during logo / carousel / film generation
  uploading.json    - shown during photo upload
  connecting.json   - shown when reconnecting to backend

If a file is missing, the app falls back to a matching hand-coded
Flutter animation, so the app still looks polished out of the box.

Recommended source: https://lottiefiles.com (search for "loading",
"generating", "upload" - free tier plenty for a brand app).
